# VVIT Time Table Structure

This repository contains the time table structure for various departments at VVIT. The folder structure is organized as follows:

- **VVIT_TimeTable_All_Departments**: Contains folders for each department.
  - **ALL_DEPARTMENTS_COMBINED**: A combined folder for all departments.
  - **CSE**: Contains time table for Computer Science Engineering.
  - **CSE-AIML**: Contains time table for CSE with special focus on Artificial Intelligence and Machine Learning.
  - **ECE**: Contains time table for Electronics and Communication Engineering.
  - **EEE**: Contains time table for Electrical and Electronics Engineering.
  - **MECH**: Contains time table for Mechanical Engineering.
  - **CIVIL**: Contains time table for Civil Engineering.
  - **IT**: Contains time table for Information Technology.
  - **AIDS**: Contains time table for Artificial Intelligence and Data Science.
  - **CSBS**: Contains time table for Computer Science and Business Systems.
  - **CSO**: Contains time table for Computer Science and Operations.
  - **CIC**: Contains time table for Computer Information Systems.

This structure is designed to provide clear organization and easy access to department-specific time tables.
